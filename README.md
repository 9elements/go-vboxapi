# go-vboxapi
[![GoDoc][godoc]](https://godoc.org/github.com/blacktop/go-vboxapi)
![License][license]  

Go interface to VirtualBox's remote [API](https://www.virtualbox.org/sdkref/)

### MVP
 - [ ] PowerUp (from snapshot)
 - [ ] PowerDown
 - [ ] List all VMs
